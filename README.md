Global Economic Indicators Analysis (2000-2024)

This project focuses on transforming fragmented global economic data into a centralized, interactive decision-making tool. By processing over 21,000 rows of data from the World Bank, I built a dynamic dashboard that allows for real-time comparative analysis of macro-economic health across 217 countries.

Core Metrics Analyzed

 -GDP Growth (Annual %): Tracking economic expansion and recession cycles.
 
 -Inflation (Consumer Prices): Visualizing purchasing power and price stability.
 
 -Unemployment Rate: Analyzing labor market health across different regions.
 
 -Official Exchange Rates: Monitoring currency fluctuations against global benchmarks.
 

Technical Workflow

 1-Data Extraction & ETL: Utilized Power Query to clean and normalize raw CSV datasets. Handled missing values and standardized country codes for consistency.
 
 2-Data Modeling: Structured the workbook into a 3-tier architecture: Data (Raw), Analysis (Pivot logic), and Dashboard (UI).
 
 3-Dynamic Visualization: Implemented advanced Excel features including:
 
   Dynamic Chart Titles: Using cell references to update titles based on Slicer selection.
   
   Interactive Slicers: Allowing users to toggle between countries and indicators instantly.
   
   Clean UI Design: Removed gridlines, headers, and field buttons to mimic a standalone software application interface.

Key Insights

 -Volatility Analysis: The dashboard clearly highlights the impact of the 2008 Financial Crisis and the 2020 Global Pandemic across diverse economies.
 
 -Currency Correlation: Comparative views reveal how exchange rate fluctuations often precede shifts in inflation rates for emerging markets.
